Keep AI coding context fresh without manual prompt wrangling. awesome-context-engine initializes repository memory, scans existing repositories to baseline context, indexes your project, syncs compact context files, and generates Clean Commit messages for tools like Copilot, Claude, Cline, Continue, Cursor, and Codex.

Quick Start

Run the CLI from your repository root and follow the guided setup.

  1. npx awesome-context-engine init to initialize context and AI integration files.
  2. npx awesome-context-engine scan to baseline repository context for existing codebases.
  3. npx awesome-context-engine index to refresh project structure mapping.
  4. npx awesome-context-engine sync to regenerate compact AI context.
  5. npx awesome-context-engine auto to watch and update continuously.

Special Commands

Use these advanced commands for communication-ready outputs from your repository history.

  • npx awesome-context-engine commit-msg to generate a Clean Commit-ready title and description.
  • npx awesome-context-engine commit-msg --breaking to mark incompatible changes when valid.
  • npx awesome-context-engine eod-report 2026-04-24 to produce a date-based leadership summary.
  • npx awesome-context-engine eod-report 2026-04-24 --json --compact for dashboards and automation pipelines.
  • npx awesome-context-engine scan --dry-run --verbose to preview baseline context updates with no file writes.

End Of Day (EOD) Reports

Turn your daily commits into a leadership-ready report with executive summary, delivery stats, and detailed plain-English outcomes.

  • npx awesome-context-engine eod-report 2026-04-24 for readable bullet output.
  • npx awesome-context-engine eod-report 2026-04-24 --json --compact for automation and dashboards.

Auto-start on Reopen

During setup, init performs a baseline repository scan automatically, so context files are ready immediately. After setup installs the VS Code folderOpen task, you do not need to manually run auto each time. Reopening the workspace starts the index + sync cycle automatically. One-time action: set this workspace to Allow Automatic Tasks from Tasks: Manage Automatic Tasks in Folder.

Clean Commit Standard

Commit message generation follows the Clean Commit convention inspired by wgtechlabs/clean-commit. Use npx awesome-context-engine commit-msg --breaking only for incompatible changes.
