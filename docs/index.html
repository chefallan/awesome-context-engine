Keep AI coding context fresh without manual prompt wrangling. awesome-context-engine initializes repository memory, indexes your project, and continuously syncs compact context files for tools like Copilot, Claude, Cline, Continue, Cursor, and Codex.

Persistent Memory
Store reusable project context in .awesome-context.
Auto Sync
Watch file changes and keep context fresh with minimal effort.
Token Efficient
Generate compact, high-signal context for AI sessions.
Local-first
Run entirely in-repo without external service requirements.
Safe Defaults
Preserve existing memory files and merge integration rules safely.
VS Code Startup Task
Optional folderOpen task for automatic syncing on workspace open.

Quick Start

Run the CLI from your repository root and follow the guided setup.

  1. npx awesome-context-engine init to initialize context and AI integration files.
  2. npx awesome-context-engine index to refresh project structure mapping.
  3. npx awesome-context-engine sync to regenerate compact AI context.
  4. npx awesome-context-engine auto to watch and update continuously.

Designed for macOS, Ubuntu/Linux, and Windows with Node.js 18+.
